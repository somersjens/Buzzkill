Gentle nudges for mindful app time

Buzzkill helps you spend less time in specific apps using Apple’s Screen Time APIs. Pick the apps that tempt you, set a reminder interval, and get lightweight, on-device notifications while you’re using them—plus a nudge when your daily goal is reached. Private by design. No tracking.

Features

Select & focus

Choose apps with the Screen Time picker (FamilyControls). Buzzkill monitors only those selections on-device using DeviceActivity.

Timed nudges

Set an app-specific interval (5, 10, 15, 20, 25, or 30 minutes). Get gentle local notifications while a selected app is in use and when the daily goal is reached.

Dynamic messages

Notifications rotate through engaging prompts and include your total time in that specific app for the day. Choose a tone and language that fits you.

How it works
  1. Pick apps with the Screen Time chooser.
  2. Set intervals from 5 to 30 minutes per app.
  3. Get nudges while you’re in the app and at your daily goal.
  4. Stay private — no screenshots, no server logs, no data transmission.
Privacy at the core

On-device only

All processing stays on the device. Permissions are managed in iOS Settings.

No blocking

Buzzkill does not block apps. Start or stop reminders anytime with a single control.

No tracking

No screenshots or recordings. No analytics SDKs. No server logging. Ever.
