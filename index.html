Buzzkill

Gentle nudges for mindful app time

Buzzkill helps you spend less time in selected apps using Apple Screen Time APIs. Choose the apps, set an interval, and get on-device reminders while you use them. You also get a nudge when the daily goal is reached. Private by design. No tracking.

Features

Select and focus

Pick apps with the Screen Time picker (FamilyControls). Buzzkill monitors only those choices on device using DeviceActivity.

Timed nudges

Set an interval per app: 5, 10, 15, 20, 25, or 30 minutes. Reminders appear while the app is in use and when the daily goal is reached.

Dynamic messages

Notifications rotate through short prompts and show your total time for that app today. You can pick a tone and a language.

Privacy at the core
  • On-device only

    All processing stays on the device. Permissions live in iOS Settings.

  • No blocking

    Buzzkill does not block apps. Start or stop reminders with one control.

  • No tracking

    No screenshots, no server logs, no analytics SDKs.
